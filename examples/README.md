# Oko Examples

Starter examples for building dApps with Oko. You can run each example locally
by following the instructions in the README.

## Requirements

- Node.js 22+
- Yarn (workspaces)

## Examples

- **[Cosmos Next.js Example](cosmos_nextjs)** — Oko Cosmos SDK on Next.js
- **[EVM Next.js Example](evm_nextjs)** — Oko Ethereum SDK on Next.js
- **[EVM Wagmi Next.js Example](evm_wagmi_nextjs)** — Next.js + wagmi +
  RainbowKit with Oko
- **[Multi-Ecosystem React Example](multi_ecosystem_react)** — React + Vite with
  Oko for both Cosmos and EVM
